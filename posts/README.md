# Как добавить новый пост

1. Создайте файл `имя-поста.md` в этой папке
2. Добавьте метаданные в начале файла (между `---`)
3. Напишите контент в формате Markdown

## Шаблон поста

```markdown
---
title: Название статьи
date: 2024-01-15
author: Ваше имя
cvss: 7.5
cve: CVE-2024-XXXX
tags: [tag1, tag2, tag3]
image: https://images.unsplash.com/photo-xxx
---

Краткое описание статьи для превью (первый абзац).

## Основной контент

Пишите здесь...

### Подзаголовок

Код:
\`\`\`python
print("Hello")
\`\`\`
```

## Поля метаданных

| Поле | Обязательное | Описание |
|------|--------------|----------|
| title | ✅ | Заголовок поста |
| date | ✅ | Дата в формате YYYY-MM-DD |
| author | ❌ | Автор (по умолчанию: имя из конфига) |
| cvss | ❌ | Оценка CVSS от 0 до 10 |
| cve | ❌ | CVE идентификатор |
| tags | ❌ | Массив тегов |
| image | ❌ | URL картинки для превью |
